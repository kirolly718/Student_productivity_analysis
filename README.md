# Анализ успеваемости студентов

Веб-приложение для автоматического анализа успеваемости студентов с визуализацией данных.

## Возможности

- Загрузка данных - поддержка CSV файлов с оценками
- Автоматический анализ - расчет статистик и выявление проблемных зон
- Визуализация - графики и диаграммы для наглядного представления данных
- Детальный анализ - расширенная статистика на отдельной странице
- Логирование - запись операций для отладки и мониторинга

## Решаемая проблема

Приложение автоматизирует ручной анализ успеваемости, экономя время преподавателей и предоставляя интуитивно понятную аналитику для принятия педагогических решений.

## Технологии

- Python 3.10
- Streamlit - веб-интерфейс
- Pandas - обработка данных
- Matplotlib - визуализация
- Logging - система логирования

## Установка и запуск

1. Клонировать репозиторий:
git clone <url-репозитория>
cd student_analyzer

2. Создать виртуальное окружение:
python -m venv student_env
student_env\Scripts\activate

3. Установить зависимости:
pip install -r requirements.txt

4. Запустить приложение:
streamlit run app.py

## Использование

1. Запустите приложение - откроется браузер с главной страницей
2. Загрузите CSV файл с оценками или используйте демо-данные
3. Просмотрите статистику - средние оценки, рейтинг студентов
4. Изучите визуализацию - графики успеваемости
5. Перейдите на страницу Analysis для детального анализа

Формат CSV файла:
student,Источниковедение,Журналистика,Программирование,Менеджмент
Алексеев,4,5,5,3
Бобров,3,4,4,4

## Тестирование

Для запуска тестов выполните:
python tests/test_basic.py

## Логирование
